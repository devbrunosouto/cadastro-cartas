# 🗺️ Desafio Básico - Super Trunfo de Estados Brasileiros

## 📋 Sobre o projeto:

Este projeto é um sistema simples de cadastro de **cartas de estados brasileiros**, inspirado no jogo **Super Trunfo**. O programa permite registrar e exibir informações de dois estados, como população, área, PIB e pontos turísticos.

## ⚙️ Como funciona:

O programa segue os seguintes passos:

1. **Cadastro da Carta 1:** O usuário insere os dados do primeiro estado:
   - Nome do estado
   - Código do estado
   - Nome da cidade
   - População
   - Área (km²)
   - PIB
   - Número de pontos turísticos

2. **Exibição da Carta 1:** Os dados inseridos são exibidos na tela.

3. **Cadastro da Carta 2:** O mesmo processo é repetido para o segundo estado.

4. **Exibição da Carta 2:** Os dados do segundo estado são exibidos na tela.
